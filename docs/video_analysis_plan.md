## 四、分析方案

### 4.1 特征设计

**频域特征（平衡评估）**：
- 躯干中心（颈 + 髋中点）轨迹
- PSD 总功率 → 晃动幅度
- 低频功率占比（0–0.5 Hz）→ 平衡差的人低频主导
- 主频率

**时域特征（步态评估）**：
- Cadence（步频）— heel-strike 检测
- 膝角摆动幅度 / 峰值数
- 踝角范围

**归一化几何特征**：
- 髋宽 / 肩宽 → 对称性
- 脚间距 / 身高 → 步宽

### 4.2 建模策略

- **Pearson 相关**：单特征与 BBS 评分相关性，快速筛选
- **Ridge / Lasso 回归**：多特征回归
- **Leave-one-out CV**：3 个 subject 轮换验证（预实验，小样本可用）

### 4.3 整体流程

```
行走视频
    │
    ├── BBS 视频段（无设备）
    │       ├── 频域特征（躯干轨迹）→ 平衡能力
    │       ├── 膝/踝角时序 → 下肢功能
    │       └── heel-strike → cadence
    │
    └── pre/training 视频段（穿设备）
            └── 膝/踝角时序（无频域、无 heel-strike）

合并特征 → Pearson 筛选 → 回归建模 → 预测 BBS 评分
```
